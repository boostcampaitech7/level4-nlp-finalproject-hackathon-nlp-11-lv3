# Front-End

## 실행 방법

### 💽 &nbsp;패키지 설치
``` bash
npm install
```

### 🏃🏻 &nbsp;실행
``` bash
npm run dev
```

### ⚒️ &nbsp;빌드
``` bash
npm run build
```

## 🔢 &nbsp;Version
React `18.3.1` , react-router-dom `7.1.5` , mui `6.4.2` , axios `1.7.9`
